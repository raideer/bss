# Better SS.com

- Chrome: https://chrome.google.com/webstore/detail/bss-better-sscom/kdbfigcfjmcioeepohmfenkddijmball

## Development
### Setup
1. Install with `yarn`
2. Run with `yarn watch`
3. Load `dist` directory as unpacked extension
